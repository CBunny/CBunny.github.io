Personal Cheat Sheet

Ableton Shortcut

Keybind Function
Spacebar Start/Stop
Shift + Spacebar Start at stopped position
Tab Switch between session and arrangement
Shift + Tab Switch between track and clip view
Cmd + Opt + L/I/M/B/O/R/S Hide/Show the following:
  • L for layout
  • I for I/O
  • M for mixer
  • B for browser
  • O for overview
  • R for return
  • S for send
Cmd + T Create an audio track
Cmd + Shift + T Create a midi track
Cmd + Opt + T Create a return track
Highlight tracks then Cmd + G Group tracks
Cmd + R Rename track
Tab / Shift + Tab in Rename Move one track forward / backward
U Fold / Unfold group
Shift + ? Hide/show info view
